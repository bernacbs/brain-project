# Skill: /memory

Reviews and updates `D00 Claude/memory.md` with the current vault state.
Execute only when the user types `/memory`.

---

## Steps

1. Count current Wiki pages per subfolder in `C01 Wiki/`
2. Count files in `sources/` (pending) and `sources/processed/` (done)
3. Count active projects in `B01 Projects/`
4. Review recent changes: what was ingested, extracted, or modified since last update
5. Note recurring topics or patterns across recent ingestions
6. Flag contradictions found between Wiki pages (if any)
7. List open questions that need owner input
8. Rewrite `memory.md` with updated state, preserving the Decisions Log
